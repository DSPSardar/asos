WhatsApp Cloud API
Setup Guide

Connect your ASOS backend to Meta's WhatsApp Business Platform. This is required for the AI pipeline to receive and send messages.

⏱ ~20 minutes · Free to set up · No monthly WA fees
1
Create a Meta Developer App
⏱ 5 minRequired
1
Go to developers.facebook.com → Log in with your Facebook/Meta account
2
Click My Apps → Create App
3
Select use case: Business → Next
4
App name: ASOS - [Your Business Name] · Enter your email
5
Connect to your Meta Business Account (create one at business.facebook.com if needed)
6
Click Create App → you'll see the App Dashboard
developers.facebook.com/apps
+ Create App
After creation, find your App ID in the top left — save it, you'll need it later.
Business Verification: For production volumes (>1000 messages/day), Meta requires Business Verification. Start verification early — it can take 2–5 business days.
2
Add WhatsApp Product to Your App
⏱ 2 minRequired
1
In your App Dashboard, scroll down to Add Products to Your App
2
Find WhatsApp → click Set Up
3
Select or create a WhatsApp Business Account (WABA)
4
You'll land on the WhatsApp → API Setup page — this is where all credentials live
Test number: Meta gives you a free test phone number for development. You can send up to 5 messages to 5 verified numbers for free before going live.
3
Collect Your API Credentials
⏱ 3 minRequired

On the WhatsApp → API Setup page, collect these 4 values:

Phone Number ID Required
1234567890123456
Found under "From" section. This identifies your WA Business number in API calls.
WhatsApp Business Account ID Auto-set
9876543210987654
Used for campaign management and number registration.
Temporary Access Token Required for dev
EAAxxxxxxxxxxxxxx...
⚠ Expires in 24h. Generate a Permanent Token for production (Step 4).
App Secret Required for HMAC
App Settings → Basic → App Secret → Show
Used to verify webhook signatures. Never expose publicly.
Add to .env.production
WHATSAPP_API_URL=https://graph.facebook.com/v20.0
WHATSAPP_APP_SECRET=your_app_secret_from_basic_settings
WHATSAPP_VERIFY_TOKEN=any_string_you_choose_eg_asos_webhook_2024
Store in ASOS Settings: After deploying, go to Dashboard → Settings → WhatsApp and paste the Phone Number ID + Access Token there. They're encrypted at rest.
4
Generate a Permanent Access Token
⏱ 5 minProduction only
Critical: The temporary token expires every 24 hours. For production you MUST create a System User token or it will stop working overnight.
1
Go to Meta Business Manager → business.facebook.com
2
Business Settings → Users → System Users
3
Click Add → Name: ASOS API · Role: Admin
4
Add Assets → Select your WhatsApp Business Account → Full Control
5
Click Generate New Token → Select your App → Grant permissions:
Required Permissions:
whatsapp_business_messaging whatsapp_business_management
6
Copy the generated token → save it immediately (shown only once) → paste into ASOS Settings
5
Configure Webhook in Meta Console
⏱ 3 minRequired
1
App Dashboard → WhatsApp → Configuration
2
Under Webhook, click Edit
3
Callback URL — paste your server URL:
Callback URL
https://api.yourdomain.com/webhooks/whatsapp
Replace yourdomain.com with your actual domain. Must be HTTPS.
Verify Token
same value as WHATSAPP_VERIFY_TOKEN in .env.production
Must match exactly — case sensitive.
4
Click Verify and Save — Meta will call your server and expect your verify token back
5
Under Webhook Fields, click Manage → subscribe to:
Subscribe to these webhook fields:
messages ✓ message_status_updates ✓
Test verification manually
curl "https://api.yourdomain.com/webhooks/whatsapp\
  ?hub.mode=subscribe\
  &hub.verify_token=YOUR_VERIFY_TOKEN\
  &hub.challenge=test123"
# Expected response: test123
6
Set Up Click-to-WhatsApp Meta Ads
⏱ 5 minFor Meta Ads flow

This is how ASOS captures campaign attribution — each ad click carries Meta ad IDs that get linked to the lead automatically.

1
Go to Meta Ads Manager → Create Campaign
2
Objective: Leads → Ad Format: Click to WhatsApp
3
In Ad Creative → WhatsApp number → select your WABA phone number
4
Set Pre-filled message — this is what users send when they click your ad:
Pre-filled WhatsApp message (suggested)
Olá! Vi o anúncio de vocês e quero saber mais informações.
ASOS's AI reads this first message and begins qualification immediately. Keep it natural.
Attribution is automatic: When a user messages via a Click-to-WhatsApp ad, Meta's webhook payload includes referral.ads_campaign_id, ads_adset_id, and ads_ad_id. ASOS's conversation worker extracts these and links them to the lead automatically — no extra config needed.
