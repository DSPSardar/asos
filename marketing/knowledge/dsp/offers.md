# DSP Offers — AI Agents Bootcamp

This file is the single source of truth for what's being sold. Agents must not invent
prices, dates, or seat counts beyond what's here.

## Format

**1-week cohort.** New batch starts every Monday. ~30 seats per batch.

| Day | Session | Theme |
|---|---|---|
| Mon | 9–10 PM PKT, live on Zoom | D1 — Idea → Agent |
| Tue | 9–10 PM PKT, live on Zoom | D2 — Job Description |
| Wed | 9–10 PM PKT, live on Zoom | D3 — Knowledge & Memory |
| Thu | 9–10 PM PKT, live on Zoom | D4 — Tools & Claude Code |
| Fri | 9–10 PM PKT, live on Zoom | D5 — Test / Secure / Ship |
| Sat | Capstone work (build + deploy) | Students build their own agent |
| Sun | Capstone showcase | Live demos of deployed agents |

Core formula, taught and reinforced daily: **Agent = Claude + Job Description + Tools + Loop**.

## How students build ("Vibe Coding")

No-code AI agent building using **Claude.ai and Anthropic Console only**, entirely in the
browser. No Python, no API coding required from students. Never describe the bootcamp as
teaching Python/API development — that overstates the technical bar and misleads prospects.

## Audience

Pakistani + diaspora learners (UK, US, Canada, Gulf). WhatsApp-first market — assume DMs and
WhatsApp are the primary conversion channel, not email.
